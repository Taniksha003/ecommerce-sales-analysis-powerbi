# E-commerce Sales, Funnel & Customer Analysis

##  Project Overview
This project analyzes e-commerce sales, customer behavior, and order fulfillment performance using the Olist E-commerce dataset.
The goal was to build a business-focused Power BI dashboard that helps stakeholders understand revenue trends, customer patterns, product performance, and operational efficiency.

##  Tools and Technologies Used 
- Power BI
- Data Modeling(Star Schema)
- DAX

## Dataset Description
The analysis is based on multiple tables from the Olist dataset:
- Customers
- Orders
- Order Items
- Products
- Payments

Proper relationships were created between fact and dimension tables to ensure accurate aggregation and analysis.
